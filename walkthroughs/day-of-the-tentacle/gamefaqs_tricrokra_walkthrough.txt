+------------------------------------------------------------------------------+
| Introduction to this FAQ and the game                                | IDX01 |
+------------------------------------------------------------------------------+
Day of the Tentacle is one of the good old adventure games of long lost times. 
If you wonder how to get this game, I cannot help you, I still got my copy from
times this was still in store. I'm sure there are sites you can download it 
from, but since I cannot support software piracy, I cannot give you links.
If you can give me sites on which this game can be bought/downloaded from
WITHOUT VIOLATING COPYRIGHTS mail me, and I'll add those to this FAQ.

Also note that since this game is old you may NOT be able to play it on modern 
machines. You may need an emulator to play it.
One of the best out there is DOSBox.
Download it from: http://www.dosbox.com

I recommand only to use this FAQ when you are stuck, or else this game will be
no challenge at all. If you are stuck look at the index on what you are doing
and look at the IDX code. Open a search window (Ctrl-F in Windows and Linux and
Apple-F in MacOS).

There are a few things you should know though.
- All items you pick up can be dropped anywhere you want. Please note that in 
  most cases you can come back for those items. However there is one point in
  which you need to check before continuing or else the game can NOT be 
  completed.
- There are jewels everywhere across Kyrandia. They appear at random so I cannot
  tell you where to look for those. Only a few specific ones can be found at
  fixed places. You can do well, in collecting those, but since you can only
  carry 10 items, you may want to store them all in one location as long as the
  game allows it. There are only two tasks in which these jewels have value
- This game has a dozen of areas or &quot;rooms&quot; as we call them in adventure 
  programming terms. Each time you get into a new room I start with &quot;-&quot; and the
  name of the room, so you can check if you are in the right room.

One important note.
Day of the Tentacle is a game in which many events go in some kind of a choas
through each other. Writing a FAQ on it that takes everything one by one is 
therefore hard. In order to give some possibilities to look things up, I 
provided a Q&amp;A chapter in which I try to answer some questions. 

Another thing is, the game starts with Bernard as playable character, lateron
Hoagie and Laverne will become available. When I call a name, it means you need
to switch to that person. You can also give items to another character by 
selecting that item, and click with it on a person's face (in the switch menu)
with it.

Well, that's all folks...
Let's get started on this

- Use the mouse hole to get into the next room
- Wait till you grow again
- Pick up the bowling ball
- Leave the room (the tentacle will be gone)
- Go to the secret lab
- Use the bowling ball on the tentacles (STRIKE!!!!)

Q: How to get Laverne out of the tree?
A: Get yourself a can of red paint and use it on the Cherry Tree (Hoagie's time)
   Then you have have George Washington to chop it down, and that tree happens
   to be the same one that Laverne get stuck in. Time has been changed and
   Laverne will be out of the tree.
   
Q: How to steal the mouse toy from the cat?
A: On the top floor of the mansion in Hoagie's time. Use both bed, then swap
   them with each other. Then use the beds again and the cat will be distracted.
   Now to steal the mouse toy.
   
Q: How to disguise Laverne as a tentacle?
A: You need the Tentacle Anatomy chart from the docter's office. Send it to 
   Hoagie and let Hoagie put in on the table of the flag designer to change the
   design of the American flag. With the crank (from Bernard's time) Laverne
   can retreive it.
   
Q: How to enter the human show?
A: - First you need a name tag from the blue Tentacle in the corridor.
   - Next you need a human. You can use the mummy from the second floor.
     Use the roller skates to get him down in the lobby.
   - Put the name tag on him and you entered the show.
   
Q: How to win the human show?
A:  First you need to get rid of Harrold, since he'll win everything.
   Put the fake barf on the floor and he'll be disqualified.
   - Best Hair: Put the spaghetti on the head of the mummy and use the fork
                to comb it.
   - Best smile: Put the fake teeth from the hose on it
   - Beest laugh: The laughing box from Oozo the clown does this.
   
Q: How to get the combination from Dr. Fred?
A: Put him asleep with Decaff. He'll opent he save and a security cam
   records the combination
   
Q: All fine, but Nurse Edna wouldn't let me look at it.
A: Kick her away.

Q: That's nice, but she keeps coming back
A: Have Hoagie to put the left handed hammer on the peddestal of the
   statue maker and the statue will change. Now Nurse Edna cannot stop 
   herself from being kicked away.
   
Q: How can I save Dr. Fred from those secret agents?
A: - PGo aint the mummy red. 
   - Untie Fred
   - Use the rope to get the mummy in the in the room on the bed.
   - Use the rope to get Fred down and you'll be in the office.

Q: How can I wake Dr. Fred?
A: By giving him coffee. Put a funnel in his mouth and pour it in.

Q: How to get rid of that suicidal sucker?
A: Give him the letter from the mailbox in Hoagie's time.

Q: Is it true that you can play &quot;Maniac Mansion&quot; in DOTT?
A: Yes. Quite true. You can do that with the computer in Weird Ed's room.

Q: Ok. Dr Fred signed the contract. How to make millions of it?
A: Steal a stamp from Weird Ed. You can do so by ruining his stamp collection
   with disappearing ink. Then put the stamp on the contract and let Hoagie
   send it by putting it into the mailbox. The Pony Express will take it
   and the millions are made.
   
Q: How can I order the diamond?
A: You get the phone number from the TV in the room with the sleeping guy.
   Call and get your diamond ordered.
   
Q: The girl at the other side of the phone refuses my order. Why?
   Two possible reasons:
   Reason 1: You don't have the bank book. Get it from the table
   Reason 2: You don't have the millions yet. The previous questions can
   tell you hot to get that.
   


Q: Laverne needs a hamster. But I cannot put it in the Chron-O-John.
   How am I supposed to get it there.
A: Put it in the freezer. Laverne will be able to get it out.


Q: I need two dimes. I got one, but how do I get the other
A: You need the crowbar to get it loose. Then you can pick it up. 
   Chew the gum to release the dime (YUCK).
   
Q: And how do I get this crowbar?
A: Get the keys from the door in the room with the sleeping man. Give those
   to the thief and he'll give you the crowbar.
   
   
Q: How do I get the sweater?
A: Get the sleeping guy of the bed. Two dimes in the slot beside his bed will
   take care of that.
   
Q: How to get the sweater to Laverne?
A: Put it in the dryer and put you highload of quarters in it. (You get it
   when use use the crowbar on the candy machine). When Laverne comes in 
   the room, the dryer just finished it's program.
   
Q: The hamster fled. How to get it?
A: Use the vacuum cleaner

Q: How does it come there
A: Put the ad in Bernard's time in the suggestion box of Hoagie's time.


Q: What to do with the cat in Laverne's time?
A: Put Booboo-B-Gone on the fence. Then lure it with the toy mouse.
   Now you can use it to scare the Edisons away from the kennel
   
Q: The kennel guard won't let me.
A: Give him the dinner coupon you won at the human show and he'll leave.

Q: How can I get that freaky moustached tentacle to leave the clock alone?
A: Scare the Edisons out of the kennel with your cat


Q: Powering up Laverne's Chrono-O-John at last?
A: If you reached the lab, you need the hamster to power the machine.
   The extention chord to get the power to the unit. When the hamster flees
   use the vaccuum cleaner to get it back.

Q: How to get the lighter?
A: Switch the lighter gun with the flag gun

Q: How to get an exploding cigar?
A: After switching the guns, ask for one.

Q: How to get that golden feather?
A: Give Washington an exploding cigar. Then give him the fake teeth.
   Now steal the blanket and put in on the chimney. The man will flee and
   you are free to grab the pen.
   
Q: How do I get vinegar?
A: Give a bottle of wine to the guy with the time capsule. Once you done
   that have Laverne to find that capsule and open it with the can opener
   to get it.
   

Q: How to get to talk to Ben Franklin?
A: First get him inside. Wash the carriage to do that

Q: How to get water and soap?
A: Bucket and water in the kitchen. The soap from the cleaning cart.
   That cart will appear if you mess up George Washington's bed and call for
   the cleaner with the chord in that room
   
Q: Red doesn't allow me to take the lab coat
A: Get the Help-wanted-sign from Bernard's time and show it to him.
   This will get you the coat.
   
Q: How to power the super-battery
A: Give the labcoat to Ben Franklin, put the battery in it and push it
   when you are asked to.

function faq_search(id, term)
{
	var term = $('#faq_search_input').val();
	var len = term.length;
	if(term.length<3)
		return;

	$.ajax(
	{
		type: 'POST',
		url: '/ajax/faq_search',
		data:  {id: 52988, term: term, key: 'b2a6cc0c'},
		success: function(response)
		{
			if(response[0]=='error')
			{
				$('#results_table').html('<tr><td colspan="2" class="dismiss_results"><span onclick="dismiss_faq_search_results(0);">No Results Found - Close</span></td></tr>');
				$('.close_results').remove();
				$('#faq_search_results').show();
			}
			else
			{
				window.currentpage = 0;
				$('#results_table').html('');
				$('.close_results').remove();

				$.each(response,function(j,i)
				{
					var val = i.value;
					val = val.split('-');
					var page = val[0];
					
					if(window.currentpage!=page)
					{
						window.currentpage = page;
					}
					var link, pagenum = '';

					
					
					{
						link = "search_highlight(52988,"+val[0]+",2,"+val[1]+","+len+");";
					}
					

					$('#results_table').append('<tr data-url="'+link+'" data-row="'+j+'" style="cursor:pointer;" onclick="search_result('+j+');"><td class="title">'+i.label+'</td><td class="type">'+pagenum+'</td></tr>');
				});

				$('#results_table').after('<div onclick="dismiss_faq_search_results(0);" class="close_results">Close Results</div>');
			}
		}
	});
}

function set_bookmark(auto)
{
	var pos = $(document).scrollTop();
	if(pos!=window.pos)
		window.pos = pos;
	else
		return;
	var page = 0;
	if(page!=window.page)
		window.page = page;
	if(auto)
		auto = 1;
	$.ajax(
	{
		type: 'POST',
		url: '/ajax/faq_bookmark',
		data: { id: 52988, pid: 564903, pos: pos, page: page, section: '', auto: auto, key: 'b2a6cc0c' },
		success: function(response)
		{
			if(response)
			{
				if(response=="Missing cookie.")
					window.autobmstop = true;
				$('.faq_bookmark_jump').fadeOut();
				$('#faq_bookmark').addClass('active');
				var html = $('.bm_jumper_inner').html();
				$('.bm_jumper_inner').animate({'opacity':0},200,function()
				{
					if($('.faq_bookmark_jumper:not(.active)'))
						$('.faq_bookmark_jumper').css('visibility','visible');
					$(this).html('Bookmark Saved').animate({'opacity':1},200);
					$('.faq_bookmark_jumper').animate({'opacity':1},200);
				});
				setTimeout(function()
				{
					if($('.faq_bookmark_jumper:not(.active)'))
					{
						$('.faq_bookmark_jumper').animate({'opacity':0},200,function()
						{
							$(this).css('visibility','hidden');
						});
					}
					$('.bm_jumper_inner').animate({'opacity':0},200,function()
					{
						$(this).html($.trim(html)).animate({'opacity':1},200);
					});
				}, 3000);
			}
		}
	});
}
